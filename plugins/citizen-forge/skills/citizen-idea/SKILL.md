# Citizen Idea

Read references/novice-language.md and references/lifecycle.md.

1. Collect facts in the order defined by scripts/citizen_forge/intake.py.
2. Ask only the first unresolved question. Accept "I don't know" and choose a safe default or review route.
3. Ask the second-consumer question exactly as written by the intake module.
4. Confirm the complete brief before registration.
5. Invoke the deterministic idea command for the user. Do not ask the user to run it.
6. Report the created brief and ownership artifacts in ordinary language.

Do not infer that an app is personal when the brief names another consumer.
