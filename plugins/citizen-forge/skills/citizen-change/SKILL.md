# Citizen Change

Classify before editing. Compare the request with the confirmed brief. Re-triage any risk-fact change.

Routine changes may proceed only when all required checks pass. Destructive migrations, infrastructure, auth, external data, dependencies, network destinations, secrets, policy, road or agent constraints, exposure, sensitivity, autonomy, irreversibility, ownership removal, and disabled backup or audit are consequential.

Stop for the required human gate. The creator cannot self-approve a high-risk production change.
