# Lifecycle

Citizen Forge moves applications through deterministic states. AI may explain or recommend a transition, but the state machine decides whether it is allowed and records evidence in the audit chain.

Routine order: idea, confirmation, registration when shared, triage, approved road, provisioning, building, release candidate, running, governed changes, archive, and retirement.

Prototype status ends as soon as another person opens, runs, depends on, or directly uses the application.
