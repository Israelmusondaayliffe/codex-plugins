# Agent operations routes

- agent-design: define a reusable agent, its contract, tools, authority, and verification.
- audit: assess an existing agent system without changing it.
- loopkit-handoff: route generic Codex Goals, bounded loops, execution, verification, resume, schedules, and runtime diagnosis to `loopkit:loopkit`.

Outcome Engine owns general outcome delivery. LoopKit owns generic Codex Goals and loops. Agent Ops owns reusable agent-system design, routing, and audit.
