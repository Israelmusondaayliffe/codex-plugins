# Agent operations routes

- agent-design: define a reusable agent, its contract, tools, authority, and verification.
- goal-design: define a sustained bounded outcome and evidence surface.
- goal-run: execute an existing Goal contract.
- bounded-loop: repeat inspection and action until a named stop condition.
- audit: assess an existing agent system without changing it.
- resume: continue an existing run from durable state after checking that its contract is still valid.

Outcome Engine owns general outcome delivery. Agent Ops owns the design and operation of agents, Goals, and loops.
