# Source Notice

Outcome Engine is an original general-purpose adaptation informed by the following public sources:

- Matt Pocock, "5 Agent Skills I Use Every Day": https://www.aihero.dev/5-agent-skills-i-use-every-day
- Matt Pocock's `skills` repository: https://github.com/mattpocock/skills
- Source skills studied: `grill-me`, `to-prd` and its current `to-spec` form, `to-issues` and its current `to-tickets` form, `tdd`, and `improve-codebase-architecture`.

The source repository is published under the MIT License. Outcome Engine does not copy its software-specific workflow as-is. It preserves the core methods while replacing mandatory codebase, GitHub, issue-tracker, and software-test assumptions with domain-neutral sources, briefs, proof surfaces, work slices, and system reviews.

Outcome Engine is a local instruction plugin. It does not add a network service or collect data by itself. The manifest's website points to the authenticated developer's GitHub profile. Its linked privacy and terms pages describe the GitHub service that hosts that profile, not a separate Outcome Engine service.
