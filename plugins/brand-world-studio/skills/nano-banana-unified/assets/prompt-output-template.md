Use for every generated set:

Mode: CREATE, EDIT, BRAND, or GRID

Distribution:
- SAFE: 1
- INVENTIVE: 3
- BOLD: 3
- EXPERIMENTAL: 3

Validation:
- Exactly 10 prompts unless the user explicitly overrides.
- Aspect ratio present when no image is uploaded.
- Uploaded images described explicitly when used.
- No MidJourney parameters.
- No em-dashes.
