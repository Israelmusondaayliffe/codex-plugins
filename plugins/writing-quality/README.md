# Writing Quality

Writing Quality packages an intent-aware writing workflow for drafting, rewriting, detect-only review, claim boundaries, and final validation.

## Owned skills

- writing-quality-router
- claim-boundary-checker
- business-writing-intent-enforcer
- writing-enforcer

## Companion capabilities

- Knowledge Work Superpowers for evidence-led analysis and deliverables
- Documents or Google Docs for file and collaborative document work

## Boundaries

- Detect-only requests do not authorize rewriting.
- Style work does not authorize new factual claims.
- Humanizer and stop-slop remain loose skills until a parity review establishes that they add distinct value.

## Verification

Run scripts/verify_bundle.py from any directory. Installation is trusted only after plugin validation, source-to-cache parity, live listing, and clean-task discovery all pass.
