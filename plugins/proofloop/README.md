# ProofLoop

ProofLoop provides bounded task execution, explicit verification, quarantined
learning records, governed memory review, and read-only audits.

## Install

    codex plugin add proofloop@israel-codex-plugins

## Included skills

- proofloop-run
- proofloop-memory-review
- proofloop-audit

ProofLoop skills are explicit-use capabilities. Start a new Codex task after
installation.
