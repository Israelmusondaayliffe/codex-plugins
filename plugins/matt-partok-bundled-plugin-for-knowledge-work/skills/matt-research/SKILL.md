# Matt Research

Answer one decision-blocking question. Use official documentation, specifications, source code, first-party datasets, and other primary sources whenever available.

Delegate to a background agent when current instructions permit it and parallel work saves time. Otherwise research in the active thread.

1. State the question, decision it informs, and stop condition.
2. Inspect existing local sources before public search.
3. Trace each material claim to the source that owns it.
4. Separate sourced fact, inference, uncertainty, and recommendation.
5. Save findings in the user-approved durable workstream location, following the closest project convention.
6. Cite each material claim near the statement it supports.
7. Return the answer, unresolved uncertainty, artifact path, and the next decision now unblocked.

For broad research packages, source ledgers, or claim-ledger drafting, hand off to Knowledge Work Superpowers instead of expanding this bounded ticket.
