# Video production routes

- faceless-explainer: narrated informational video without an on-camera host. Owner: faceless-explainer.
- product-launch: product announcement, feature reveal, or commercial launch asset. Owner: product-launch-video.
- pr-story: turn a pull request or technical change into a visual explanation. Owner: pr-to-video.
- website-capture: capture and present an existing website or web product. Owner: website-to-video.
- music-visualization: visuals synchronized to supplied music. Owner: music-to-video.
- slideshow: still-image or presentation-led video. Owner: slideshow.
- motion-graphics: typography, diagrams, charts, or designed animation. Owner: motion-graphics.
- general-video: mixed production not owned by a specialist route. Owner: general-video.

Supporting skills:

- embedded-captions for burned-in captions.
- graphic-overlays for lower thirds, callouts, frames, and labels.
- media-use for source media inspection and handling.
- video-prompt-builder for model-ready generation prompts.
- remotion-to-hyperframes for an explicit migration.

Default runtime: HyperFrames. Remotion remains a companion for explicit Remotion tasks and existing Remotion projects.
