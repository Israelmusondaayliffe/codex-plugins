# Video Production Studio

Video Production Studio coordinates the full path from brief to verified video delivery.

## Owned skills

- video-production-router
- video-delivery-qc
- embedded-captions
- faceless-explainer
- general-video
- graphic-overlays
- media-use
- motion-graphics
- music-to-video
- pr-to-video
- product-launch-video
- remotion-to-hyperframes
- slideshow
- video-prompt-builder
- website-to-video

## Companion capabilities

- HyperFrames as the default implementation runtime
- Remotion for explicit Remotion work and existing Remotion projects
- Browser for website capture and rendered inspection
- Computer Use for native application steps that lack a structured surface

## Boundaries

- Loose HyperFrames base skills remain outside this plugin until runtime ownership is settled.
- The router chooses one primary production path before asset generation.
- A rendered file is not complete until technical and visual delivery checks pass.

## Verification

Run scripts/verify_bundle.py, validate the router and QC scripts, and inspect at least one real media file before relying on the delivery workflow.
