# Multi-Direction Explorer

Tier: 2 ideation and selection skill.

Use this skill to create meaningful variation from one seed.

## Router Logic

- Generate options: load `agents/agent-direction-generator.md`.
- Select or narrow options: load `agents/agent-selector.md`.
- Local option file: optionally run `scripts/check_option_count.py <file> <minimum>`.

## Required References

- `references/variation-axes.md`: reusable axes for creating real variation.

## Required Assets

- `assets/options-matrix-template.md`: reusable option comparison shape.

## Workflow

1. Identify the axis of variation: audience, tone, format, risk level, medium, business goal, visual language, or technical approach.
2. Generate at least 10 distinct directions when the user asks for breadth.
3. Make each option meaningfully different, not a synonym swap.
4. Label the logic behind each direction in plain language.
5. Recommend the top 2 or 3 when a decision is needed.
