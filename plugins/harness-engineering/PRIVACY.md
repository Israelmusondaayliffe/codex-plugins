# Privacy

Harness Engineering runs locally and sends no telemetry to its publisher.

Audit artifacts must contain paths, file hashes, configuration key names, capability names, and validation results only. They must not contain credential values, access tokens, cookies, private connector payloads, or complete environment variables.

If the user chooses to call a connector, web service, model API, or public repository, that system's privacy terms apply to that separate action. Harness Engineering requires explicit approval before authentication or public publication.
