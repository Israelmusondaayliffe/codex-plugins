# Prompt migration controls

Check the prompt job, input contract, output contract, tool policy, effort or reasoning controls, formatting, refusal behavior, and evaluation criteria.
Remove instructions that only compensated for the source model when the target model does not need them.
Do not copy unsupported parameters or assume names stayed stable across ChatGPT, API, Codex, and third-party surfaces.
Keep source facts and user constraints unchanged unless the user authorizes a product change.
