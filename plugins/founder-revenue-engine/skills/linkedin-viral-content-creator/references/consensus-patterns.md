# Consensus Patterns

Pre-mapped consensus takes for common niches to accelerate Phase 1 execution.

## AI/Technology

**Boring Takes (p > 0.90):**
1. "AI will transform everything" - Vague, no specific insight
2. "The key is prompt engineering" - Generic skill advice
3. "AI is a tool, not a replacement" - Defensive platitude
4. "We're in the early days of AI" - Timeline observation, no position
5. "AI will augment human creativity" - Aspirational without mechanism

**Vocabulary Trap:**
transform, leverage, augment, disrupt, revolutionary, game-changer, paradigm shift

**Boredom Diagnosis:**
Focuses on potential while avoiding current economic reality, labor displacement, and structural power shifts.

## Usage Notes

When running Phase 1 for any of these niches:
1. Start with the pre-mapped boring takes
2. Customize based on user's specific sub-niche
3. Add niche-specific vocabulary traps
4. Update boredom diagnosis with current context

For unlisted niches:
1. Identify the 5 most repeated talking points
2. Find the aspirational language everyone uses
3. Locate the structural reality being ignored
4. Map what consensus optimizes for (individual vs system, aspiration vs reality)
