# Overlap states

- identical-mirror: the name and SKILL.md fingerprint match.
- drifted-copy: the name matches but the fingerprint differs.
- namespaced-bundle: a plugin intentionally contains a copy of a loose skill.
- trigger-collision: different names claim substantially the same user intent.
- superseded: an explicit owner and migration decision identifies the replacement.
- unresolved: evidence does not establish the relationship.

Never infer supersession from recency alone.
