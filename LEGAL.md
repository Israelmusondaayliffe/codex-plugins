# Legal and licensing notes

This repository is a distribution collection, not a blanket relicensing of
every file under one license.

## Package-specific terms

- knowledge-work-superpowers includes its own MIT LICENSE and NOTICE.md.
- outcome-engine declares MIT in its plugin manifest and includes NOTICE.md.
- loopkit declares MIT in its plugin manifest and includes a LICENSE file.
- harness-engineering declares MIT in its plugin manifest and includes a LICENSE file.
- Other plugins do not currently declare a top-level package license. Copyright
  remains with the relevant author unless a file or package states otherwise.
- Third-party notices and license files inside plugin folders remain in force.

Review the target plugin before copying, modifying, or redistributing it. Public
repository access does not erase package-specific terms.

## Excluded asset

The local CyberpunkReplica.ttf file used by one embedded-captions theme carries
a personal-use license and is not distributed here. The theme generator accepts
CYBERPUNK_REPLICA_FONT_PATH so users with an appropriately licensed copy can
provide it locally.

## External services

Some scripts can interact with public APIs or third-party services. Their terms,
rate limits, authentication requirements, and content policies apply separately.
Users are responsible for configuring and using those services lawfully.
